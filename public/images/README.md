# Képek mappája

## Használat:
1. Tedd ide a feldolgozandó képeket
2. Használd az AI Asszisztenst az elemzéshez
3. A rendszer automatikusan a megfelelő mappába menti a feladatokat

## Mappa struktúra:
- `public/images/` - Eredeti képek
- `exercises/[osztály]/[tantárgy]/` - Feldolgozott feladatok

## Támogatott formátumok:
- JPG, PNG, WebP
- Ajánlott méret: max 2MB
